Detailed Description

Exponential Moving Average.

This filter can be seen as an integrator:

with a reset period acting as anti-windup. It can also (informally) be interpreted as the average value of the input signal over the last seconds. Formally, it represents the amount of time for the smoothed response of a unit input to reach of the original signal.

See https://en.wikipedia.org/wiki/Exponential_smoothing. It is equivalent to a low-pass filter applied to the integral of the input signal.
